# Calculadora Matemática en Java y HTML

Este proyecto consiste en una **calculadora matemática** interactiva que permite realizar diversas operaciones matemáticas. La aplicación está compuesta por una interfaz de usuario en HTML y JavaScript para manejar las operaciones y una parte en Java para realizar los cálculos.

## Características

- La interfaz en HTML permite al usuario seleccionar diferentes tipos de operaciones matemáticas (operaciones básicas, sucesiones, orden y probabilidad).
- Para cada tipo de operación, el usuario puede elegir una suboperación específica, como "MCM", "MCD", "Secuencia Fibonacci", entre otras.
- Al seleccionar una operación y suboperación, el usuario puede presionar un botón para calcular el resultado. Los resultados se muestran en un área de texto en la interfaz.

## Funcionalidades

1. **Operaciones básicas**:
   - División
   - MCM (Mínimo común múltiplo)
   - MCD (Máximo común divisor)
   - Verificar si un número es primo

2. **Sucesiones**:
   - Secuencia Fibonacci
   - Congruencia
   - Teorema de Euler
   - Sucesión aritmética
   - Sucesión geométrica

3. **Orden**:
   - Números ascendentes
   - Números descendentes

4. **Probabilidad**:
   - Permutación
   - Combinación

## Tecnologías Usadas

- **HTML**: Para crear la estructura de la interfaz de usuario.
- **CSS**: Para estilizar la página y hacerla visualmente atractiva.
- **JavaScript**: Para gestionar las interacciones del usuario y las operaciones matemáticas seleccionadas.
- **Java**: Para procesar y realizar los cálculos detrás de las operaciones.
